# Tic-Tac-Toe

Android adaptation of the [Tic-Tac-Toe](https://de.wikipedia.org/wiki/Tic-Tac-Toe)-game, which uses [Vector-Assets](https://developer.android.com/studio/write/vector-asset-studio) for the fields.
